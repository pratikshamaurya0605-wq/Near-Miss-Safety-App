# Near-Miss-Safety-App
Web Application to avoid any hazard in any chemical plant like Godrej . It is an hazard reporting app, to avoid more critical accident and solve it as soon as possible
**Why this system built? Thought behind it**.
In many companies, most of the work is still done manually using registers, 
files, or sometimes simple Excel sheets. If any incident, its manually informed to safety 
officer which takes so much time in large company areas are written and stored physically. 
Because of this, a lot of problems are faced in daily work. It takes more time to 
write and manage records, and sometimes mistakes happen in reminding  or data entry. 
Finding old records is also difficult and time-consuming. There is always a risk of losing 
important data due to damage or misplacement of files. 
Even if computers are used in some places, the system is not fully organized, which 
still creates confusion and extra work for safety officers.
**Admin Dashboard** can control and update the hazard reported whether it is** Pending / In progress / Resolved **
**Worker Dashboard** can report any hazard using photo based hazard Reporting
